
# 01: How to generate images of handwritten digits using a Deep Convolutional Generative Adversarial Network (DCGAN). 

This tutorial demonstrates how to generate images of handwritten digits using a Deep Convolutional Generative Adversarial Network (DCGAN). The code is written using the Keras Sequential API with a tf.GradientTape training loop. 

Key Concepts
------------
- GANs
- Keras Sequential API
- tf.GradientTape

Objective
---------
Generate images of handwritten digits using a Deep Convolutional Generative Adversarial Network (DCGAN).
